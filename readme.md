# 项目描述
此项目用于将sql server 2008 r2的数据库迁移到postgresql 15。
# 迁移步骤
1. 自动连接192.168.1.111\server2008下面的pmsmasterdev,pmsdev数据库
2. 自动遍历连接数据库中的所有表，并将表结构转换为postgresql的格式，然后在postgresql 15上创建相应的表。
3. 自动遍历连接数据库中的所有存储过程，并将存储过程转换为postgresql的格式，然后在postgresql 15上创建相应的存储过程。
4. 自动遍历连接数据库中的所有视图，并将视图转换为postgresql的格式，然后在postgresql 15上创建相应的视图。
5. 自动遍历连接数据库中的所有函数，并将函数转换为postgresql的格式，然后在postgresql 15上创建相应的函数。
6. 自动遍历连接数据库中的所有触发器，并将触发器转换为postgresql的格式，然后在postgresql 15上创建相应的触发器。
7. 迁移完成后，生成一个报告文件，包含迁移的表、存储过程、视图、函数和触发器的列表，以及任何错误信息。
# 注意事项
- 确保在迁移过程中不会影响原sql server数据库。
- 提供一个界面，可以选择要迁移的源数据库和目标数据库。
